# URL Object
The object describing a contact's URL.

## Example
```js
{
    "url" : new URL( "https://www.example.com/" ).href,
    "type" : "WORK"
}
```

## Properties
1. `url` : string (optional) — url relevant to the contact.
2. `type` : "HOME" or "WORK" (optional) — the type of the contact's link.
